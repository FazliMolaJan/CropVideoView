# CropVideoView
This is a video library for android.

It has all scale types. Also user can change background of view

## How to install


``` 
repositories {
    maven { url "https://mishatronic.bintray.com/cropvideoview" }
} 
```

``` 
dependencies {
    ...
    implementation 'mishatronic:cropvideoview:0.0.1'
} 
```
